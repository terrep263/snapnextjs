## Troubleshooting

### "I don't see my free event after signing in"

**Possible reasons:**

1. **You used the wrong email**
   - The message said: `sarah@example.com`
   - You signed up with: `sarah.johnson@example.com`
   - **Solution:** Sign up/log in with the correct email

2. **You haven't confirmed your email yet**
   - Check your email for confirmation link
   - Click the link to confirm
   - **Solution:** Check spam/junk folder too!

3. **You're looking in the wrong place**
   - Your event should be on your dashboard
   - Go to: `snapworxx.com/dashboard` (after logging in)
   - **Solution:** Make sure you're logged in

### "The signup link doesn't work"

- Check if link has any typos in the email
- Copy the exact link from the email
- Make sure you have internet connection
- Try a different browser (Chrome, Safari, Firefox)
- **Contact support:** support@snapworxx.com

### "I forgot my password"

- Go to: `snapworxx.com/login`
- Click: "Forgot Password?"
- Enter: Your email
- Check email for reset link
- Create new password

### "My guests can't upload photos"

**Guest experience:**
1. Guest scans QR code or clicks link
2. Gallery appears in their browser
3. Guest clicks "Upload Photos"
4. Guest selects files from their phone/computer
5. Photos upload instantly

If not working:
- Make sure photos aren't too large (100MB max per file)
- Make sure browser has permission to access files
- Try a different browser
- Check internet connection
- **Contact support:** support@snapworxx.com
